ADELIE Construction Website v1.1

WHAT IS INCLUDED
- Responsive, mobile-friendly custom website
- Home, About, Services, Projects, Process, ADELIE Promise, Contact
- Service Areas page
- Transparency Center / homeowner education page
- Enhanced and web-optimized project photography
- LocalBusiness/GeneralContractor structured data
- Sitemap, robots.txt, canonical links and social-sharing metadata
- Accessibility improvements and mobile navigation

OPEN LOCALLY
1. Unzip the package.
2. Open index.html in a browser.

PUBLISHING
This is a static website and can be hosted on Cloudflare Pages, Netlify, Vercel, GitHub Pages or a traditional web host.
Neo can continue managing the domain and business email. When the website is published, only the website-related DNS records should be changed. Keep Neo email records—especially MX, SPF, DKIM and verification records—unchanged.

CONTACT FORM
The current form opens the visitor's email application with the project information prefilled.
Before the public launch, connect a hosted form endpoint for more reliable lead capture.

CONTRACTOR LICENSE
No CSLB number or licensed-contractor claim has been added. Add the license number after issuance and confirm the required advertising format.
