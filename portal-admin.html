Final Setup

  1. Run supabase/adelie-platform-v1.sql once in Supabase SQL Editor.

  3. Add SUPABASE_URL and SUPABASE_SERVICE_ROLE_KEY to Netlify environment variables.
  4. Deploy this folder and test a client invitation.
