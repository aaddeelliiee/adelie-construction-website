Portal Setup

This admin interface uses your authenticated Supabase account. Client invitations are sent through a Netlify Function so the private service-role key never appears in browser code.

  1. Run supabase/portal-schema.sql in Supabase SQL Editor.
  2. Create your administrator user in Supabase Authentication.
  3. Add that user's UUID to portal_admins.
  4. Set Netlify environment variables SUPABASE_URL and SUPABASE_SERVICE_ROLE_KEY.
  5. Redeploy the site and test with a client invitation.
