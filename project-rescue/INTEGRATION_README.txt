SAFE INSTALLATION
1. Add these files to the root folder of the CURRENT ADELIE website project.
2. Add a top-level navigation link labeled “Project Rescue” pointing to project-rescue.html.
3. Add a homepage feature card or section linking to project-rescue.html.
4. Rebuild/redeploy the complete website folder through Netlify.
5. In Netlify, confirm the form named “project-rescue-health-check” appears under Forms.
6. Submit one test request and confirm the notification email.
7. Add https://www.adelieconstruction.com/project-rescue.html to the main sitemap.xml and request indexing in Google Search Console.

IMPORTANT
This package is an additive module. Do not drag only this folder into Netlify as a replacement for the entire live website, because a drag-and-drop deployment replaces the prior deployment. Merge it into the current full website source first.
